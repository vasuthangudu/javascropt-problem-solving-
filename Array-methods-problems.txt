1. map()
Definition

map() creates a new array by applying a function to every element.

Syntax
array.map((item, index) => {})
Example
let nums = [1, 2, 3, 4];

let result = nums.map(num => num * 2);

console.log(result);
Output
[2, 4, 6, 8]
Small Problem
Convert all names to uppercase
let names = ["vasu", "ram", "sai"];

let upper = names.map(name => name.toUpperCase());

console.log(upper);
2. filter()
Definition

filter() returns elements that satisfy a condition.

Example
let nums = [10, 15, 20, 25];

let even = nums.filter(num => num % 2 === 0);

console.log(even);
Output
[10, 20]
Small Problem
Find students above 18 age
let ages = [16, 18, 20, 15, 25];

let adults = ages.filter(age => age >= 18);

console.log(adults);
3. reduce()
Definition

reduce() reduces array into a single value.

Example
let nums = [1, 2, 3, 4];

let sum = nums.reduce((total, num) => total + num, 0);

console.log(sum);
Output
10
Small Problem
Find total cart price
let prices = [100, 200, 300];

let total = prices.reduce((sum, price) => sum + price, 0);

console.log(total);
4. forEach()
Definition

forEach() executes a function for every array element.

Example
let fruits = ["Apple", "Mango", "Banana"];

fruits.forEach(fruit => {
    console.log(fruit);
});
Small Problem
Print squares
let nums = [1, 2, 3, 4];

nums.forEach(num => {
    console.log(num * num);
});

24. fetch() (Important)
Definition
